# rails_angular
Rails with Angular2

How to run Rails
- open terminal
- open folder railsng
- bundle install
- rails db:migrate
- rails db:seed
- rails s

How to run Angular
- copy folder railsng_frontend to root (linux : var/www/yourproject | windows : xampp/htdoc/yourproject)
- open your browser and open project address (http://localhost/yourproject)

easy right?
